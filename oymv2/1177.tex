\texte{
Soit $A = \begin{pmatrix}
2&2&0\\1&2&1\\0&2&2\\
\end{pmatrix}$. D\'eterminer les $\lambda \in \Rr$ tels que $ \exists X \in \Rr^3
-\left\{ (0, 0, 0)\right\}$ tel que $AX = \lambda X$. Pour chaque $\lambda$
d\'eterminer $E_{\lambda} = \left\{ X \in \Rr^3 / AX = \lambda X\right\}$.
}
}
