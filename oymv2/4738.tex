\texte{
\def \xxxx{\vec x}\def\yyyy{\vec y}\def\zzzz{\vec z}%
Soient $\xxxx,\yyyy,\zzzz$ trois vecteurs d'un evn $E$ tels que $\xxxx+\yyyy+\zzzz = \vec 0$.

Montrer que : $\|\xxxx-\yyyy\,\| + \|\yyyy-\zzzz\,\| + \|\zzzz-\xxxx\,\| \ge
       \frac32(\|\xxxx\,\|+\|\yyyy\,\|+\|\zzzz\,\|)$.
}
}
