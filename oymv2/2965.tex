\texte{
Soient $G,H$ deux groupes multiplicatifs. On munit $G\times H$ de l'op{\'e}ration :
$$\forall\ g,g'\in G,\ \forall\ h,h'\in H,  \qquad (g,h)\cdot(g',h') = (gg',hh').$$
\\
Montrer que $\cdot$ d{\'e}finit une loi de groupe sur $G\times H$.
}
}
