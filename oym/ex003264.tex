  \question{Soit $P = X^4 + aX^2 + bX + c$ de racines $\alpha,\beta,\gamma,\delta$.
CNS pour ces racines soient en progression arithm{\'e}tique~?}
  \reponse{$b=0,\ c = \frac 9{100}a^2  \Rightarrow $ racines : $-3x,-x,x,3x$ avec
$x = \sqrt{\frac {-a}{10}}$.}
