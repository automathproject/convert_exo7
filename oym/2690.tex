  \texte{}
  \question{Soit $f:[-a,a] \to \R$ une fonction de classe $C^2$. Montrer que
$$\forall x \in [-a, a],\qquad
|f'(x)| \leq  \frac 1{2a} |f(a) -f(-a)| + \frac{a^2+x^2}{2a}
\sup_{t \in [-a,a]} |f''(t)|.$$
Application : montrer que si $0 \leq x \leq \pi /2$, on a 
$\sin x \geq  x \cos x -x^2$.}
  \reponse{}
