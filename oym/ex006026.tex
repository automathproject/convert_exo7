monter dans l'avion, c'est-à-dire $n$ tel que : $P[X>150] \leq 0.05$ ?}
  \item \question{Reprendre le même exercice avec un avion de capacité de $300$
places; faites varier le paramètre $p = 0.5$ ; $p=0.8$.}
