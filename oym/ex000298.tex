  \texte{}
  \question{D\'eterminer l'ensemble de tous les couples $(m,n)$ tels que
$$ 955m+183n=1.$$}
  \reponse{Comme le pgcd de $955$ et $183$ est $1$, donc d'après le théorème de Bézout
cette équation a des solutions. Par exemple une solution particulière est 
$(m_0,n_0) = (-32,167)$.
Les solutions sont exactement les couples $(m,n) = (m_0 - 83k,n_0+37k)$, pour $k\in \Zz$.}
}
