  \texte{}
  \question{Soit $(ABDC)$ un parallélogramme. Déterminer les coordonnées de $D$ dans le repère $(A,\overrightarrow{AB},\overrightarrow{AC})$.}
  \reponse{Puisque $(ABDC)$ un parallélogramme, $\overrightarrow{AD}=\overrightarrow{AB}+\overrightarrow{AC}$. Les coordonnées de $D$ dans le repère $(A,\overrightarrow{AB},\overrightarrow{AC})$ sont donc $(1,1)$.}
