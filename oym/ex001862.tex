  \texte{}
  \question{Soit $ f:{\Rr}^{2}\rightarrow {\Rr}$ une application $ C^{1 } $ sur un
ouvert convexe $ O$ telle que :
$$\forall a\in O,\frac{\partial f}{\partial x_{1}}(a)=\frac{\partial f}{
\partial x_{2}}(a)=0 .$$
Montrer que $ f$ est constante sur $ O.$}
